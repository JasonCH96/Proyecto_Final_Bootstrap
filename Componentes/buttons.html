Buttons

El componente Buttons de Bootstrap se utiliza para crear botones en una página web. Los botones de Bootstrap son muy versátiles y se pueden personalizar de muchas maneras diferentes para adaptarse a las necesidades de diseño de cualquier proyecto.

Button Group

El componente Button Group de Bootstrap se utiliza para agrupar varios botones juntos en una misma sección. Los grupos de botones de Bootstrap permiten al usuario seleccionar una opción de un conjunto de opciones relacionadas.

Close Button

El componente Close Button de Bootstrap se utiliza para crear un botón de cierre en una sección de la página web, como una alerta o un modal. El botón de cierre se presenta como una pequeña X que se coloca en la esquina superior derecha de la sección.
