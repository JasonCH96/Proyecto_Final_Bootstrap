Carrousel

El componente Carousel de Bootstrap se utiliza para mostrar imágenes, contenido y otro tipo de información en un carrusel que se puede desplazar horizontalmente. Los carruseles son útiles para mostrar una cantidad de contenido limitada en un espacio reducido.
