Tables

Las tablas en Bootstrap son un componente de interfaz de usuario utilizado para mostrar datos en una estructura de filas y columnas. Las tablas son altamente personalizables y se pueden adaptar a una variedad de casos de uso, desde tablas simples hasta tablas complejas con paginación, filtros y clasificación.
