Breadcrumb

El componente Breadcrumb de Bootstrap se utiliza para mostrar una estructura de navegación en forma de migas de pan en una página web. Los Breadcrumbs de Bootstrap son útiles para mostrar al usuario dónde se encuentra en la estructura de navegación del sitio web y cómo puede regresar a las páginas anteriores.

Pagination

El componente Pagination de Bootstrap se utiliza para crear una lista de enlaces de paginación en una página web. Los Paginations de Bootstrap son útiles para mostrar grandes conjuntos de datos que se dividen en varias páginas y permiten al usuario navegar fácilmente entre ellas.
