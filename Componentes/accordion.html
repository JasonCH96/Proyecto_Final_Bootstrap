Accordion

El componente "accordion" de Bootstrap es una forma interactiva de mostrar y ocultar contenido en una página web. Al hacer clic en un elemento, se expande o contrae para mostrar u ocultar información. Es útil para organizar información en secciones y presentarla de forma compacta y fácil de navegar. Bootstrap proporciona estilos y funcionalidad predefinidos para agregarlo rápidamente a un sitio web.

Este es el cuerpo del acordeón del primer elemento. Está oculto por defecto, hasta que el complemento de colapso agrega las clases apropiadas que utilizamos para dar estilo a cada elemento. Estas clases controlan la apariencia general, así como la muestra y ocultamiento mediante transiciones CSS. Puedes modificar cualquiera de estos aspectos con CSS personalizado o sobrescribiendo nuestras variables predeterminadas. También vale la pena señalar que prácticamente cualquier HTML puede ir dentro de .accordion-body, aunque la transición limita el desbordamiento.
