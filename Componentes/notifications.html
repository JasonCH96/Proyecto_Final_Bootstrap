Badge

El componente Badge de Bootstrap se utiliza para mostrar pequeñas etiquetas de texto con información adicional en un elemento de la interfaz de usuario. Los Badges de Bootstrap son pequeños y generalmente se usan para mostrar información como notificaciones, conteos de elementos o estados. Los Badges de Bootstrap se pueden aplicar a muchos elementos de la interfaz de usuario, como botones, enlaces, iconos y otros elementos HTML.

Spinner

El componente Spinners de Bootstrap se utiliza para mostrar un indicador de carga o progreso en una página web. Los Spinners de Bootstrap son animaciones simples que indican que una tarea está en progreso y que el usuario debe esperar antes de continuar. Bootstrap ofrece dos tipos de Spinners: el tipo "border" que es un círculo con borde y el tipo "growing" que es un círculo lleno que crece y disminuye de tamaño. Los Spinners de Bootstrap también se pueden personalizar con diferentes tamaños, colores y estilos.
